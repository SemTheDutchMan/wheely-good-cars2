Put your car images in this folder.

Filename format:
- Exact match: use the car brand + model together
- Brand fallback: use only the brand name
- Remove spaces and special characters
- The lookup is case-insensitive

Examples:
- Tesla Model 3 -> teslamodel3.png
- Any Tesla -> tesla.png
- BMW X5 -> bmwx5.jpg
- Mercedes C-Class -> mercedescclass.webp

Fallback order:
1. Exact brand + model
2. Brand only
3. A random 404 image from files that start with 404, like 404.png or 404-suv.jpg
